## Modelagem de Banco de Dados: Estacionamento Inteligente

Este repositório contém o projeto de **Modelagem de Banco de Dados** desenvolvido no primeiro semestre, focado na criação de um sistema para o **Estacionamento Inteligente** de uma cidade inteligente.

O objetivo principal é projetar e implementar um banco de dados eficiente para gerenciar informações cruciais sobre vagas, veículos, otimizando a experiência do usuário e a gestão do estacionamento.

-----

### Conceitos de Banco de Dados Aplicados

Neste projeto, aplicamos e demonstramos o entendimento dos seguintes conceitos de banco de dados:

  * **Modelagem de Dados Relacional**: Utilização de chaves primárias, chaves estrangeiras e relacionamentos.
  * **Normalização**: O modelo foi projetado utilizando normalização, evitando redundância e anomalias de atualização.
  * **Integridade Referencial**: Implementada através das *Foreign Keys* para garantir a consistência dos dados.
  * **Tipos de Dados Otimizados**: Escolha apropriada de tipos de dados (e.g., `VARCHAR`, `INT`, `BOOLEAN`) para otimizar o armazenamento e performance.
