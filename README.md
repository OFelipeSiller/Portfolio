# Sobre mim
Sou estudante do 1º semestre de Ciência da Computação na UDF e estou construindo aqui minha trajetória no mundo da tecnologia. Cada projeto representa um passo da minha evolução, das primeiras linhas de código às ideias que começam a tomar forma.

## Projetos

* **Ordenação em C** – Onde aprendi, na prática, como algoritmos organizam o caos.
* **Calculadora Modular em C** – Meu primeiro exercício real de modularização e boas práticas.
* **Estacionamento Inteligente** – Um mergulho na modelagem de dados e estruturação de sistemas.
* **Barco (Web)** – Projeto em grupo que me mostrou o poder do HTML, CSS e JavaScript trabalhando juntos.

Estou explorando lógica de programação, C, banco de dados e desenvolvimento web – sempre buscando transformar desafios em aprendizado. Meu objetivo é seguir crescendo e criar projetos cada vez mais completos e criativos.
